# Vibro Lesson 1 — Учебный проект по HTML/CSS/JavaScript

## Описание
Учебный проект, созданный в рамках первого урока по веб-разработке. Демонстрирует основы создания интерактивных веб-страниц с использованием HTML, CSS и JavaScript.

## Структура проекта

```
vibro_lesson_1/
├── hello.html      # Страница с градиентным фоном и кнопкой смены цвета
├── index.html      # Страница с персональным приветствием
└── README.md       # Документация проекта
```

## Функционал

### hello.html
- Градиентный фон (фиолетово-синяя гамма)
- Белые текстовые элементы
- Кнопка со скруглёнными углами и тенью
- При нажатии на кнопку:
  - Фон страницы меняется на случайный цвет
  - Появляется всплывающее сообщение

### index.html
- Запрос имени пользователя через `prompt()`
- Персональное приветствие по имени
- Если имя не введено — приветствие «Привет, незнакомец!»
- Кнопка для повторного ввода имени

## Технологии
- **HTML5** — семантическая разметка
- **CSS3** — стилизация (градиенты, flexbox, тени, анимации)
- **JavaScript** — интерактивность (обработчики событий, DOM-манипуляции)

## Запуск
Откройте любой из файлов `.html` в браузере:
```bash
# Через файловый менеджер (двойной клик)
# или через командную строку
start hello.html
start index.html
```

## Учебные цели
- Изучение структуры HTML-документа
- Работа с CSS-свойствами и комментариями
- Основы JavaScript: функции, события, DOM
- Работа с `alert()`, `prompt()`
- Генерация случайных значений
